# Literature and Writing Agent Guidance

- source-supported facts와 interpretation을 명시적으로 분리한다.
- 직접 인용은 최소화하고 의미를 보존해 정확하게 paraphrase한다.
- 출처가 다른 정보를 하나의 사실처럼 합치지 않는다.
- 데이터셋이 제공하는 사실과 후속 논문에서 해당 데이터를 활용한 방식을 구분한다.
- 다채널 성능 향상은 실제 single-/multi-channel 비교 실험이 있을 때만 수치로 기록한다.
- 비교가 끝나기 전 UOS v2의 강점은 확정적 우위가 아니라 `candidate strength`로 표현한다.
- 모든 기술적 사실에 논문 페이지, 표, 그림 또는 절 번호를 남긴다.
- 확인되지 않은 센서, DAQ, 동시성, 결함 구성은 추론하지 않고 `Unknown`으로 둔다.
