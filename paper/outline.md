# Provisional Data in Brief Outline

## Title candidates

Evidence needed:
- 최종 데이터 구성과 검증 범위
- 검증된 차별성 및 공개 범위

## Abstract

Evidence needed:
- 최종 수집 조건, 규모, 접근 방식, 핵심 가치

## Specifications Table

Evidence needed:
- subject, data type, acquisition, repository, license

## Value of the Data

Evidence needed:
- 비교 조사로 검증된 사용자·활용 사례와 공백

## Background

Evidence needed:
- 원 출처 기반 선행 데이터셋 및 연구 필요성

## Data Description

Evidence needed:
- 파일 구조, 채널, label, metadata, 데이터 규모

## Experimental Design, Materials and Methods

Evidence needed:
- test rig, sensors, DAQ, mounting, sampling, operating/fault conditions

## Data Validation

Evidence needed:
- 회전/결함 주파수, spectrum/envelope, 반복성, 품질 관리 결과

## Limitations

Evidence needed:
- 적용 범위, rig/condition bias, label 및 측정 한계

## Ethics Statement

Evidence needed:
- 적용되는 윤리·안전·개인정보 고려사항 또는 비적용 근거

## Data Availability

Evidence needed:
- 공식 저장소, DOI, license, version, access instructions
